> 2025.04.03 v3.4.5版开始移除 java 单独版本，目前安装 java 的版本会自动升级成 py 的版本
>
> 后续版本只区分v7a和v8a：
> 	v7a兼容新旧设备，大多数设备都可以使用特点：支持更多特殊协议直播源
> 	v8a只能在新设备上安装特点：适配新硬件，在新设备上可以最大限度发挥硬件性能，运行会更高效

**关于版本：**

- mobile = 手机版
- leanback = 电视版
- java = jar + js 爬虫
- python = jar + js + py 爬虫
- arm64_v8a = 64 位元
- armeabi_v7a = 32 位元
- dev = 內测版
- kitkat = 4.x 版
- release = 正式版
